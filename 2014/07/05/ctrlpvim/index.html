vim 插件 ctrlp 常用快捷键

By Kai Guan

7月 5 2014 更新日期:7月 5 2014

文章目录
  • :CtrlP 或者:CtrlP[开始的目录]来激活插件的查找file模式
  • :CtrlPBuffer或者:CtrlPMRU来激活插件的查找buffer或者查找MRU file 模式
  • :CtrlPMixed同时激活三种模式

获得其他指令的帮助:

  • :help ctrlp-commands and :help crelp-extensions

CtrlP插件激活以后:

  • F5刷新当前目录
  • c-f和c-b来回切换三种模式
  • c-d转换文件查找匹配模式，仅匹配文件名<->匹配路径+文件名
  • c-r转换到正则匹配模式
  • c-j向下移动选择文件的箭头,c-k向上移动选择文件的箭头
  • c-t以标签页打开新文件c-v以左右分栏打开新文件c-x以上下分栏打开新文件
  • c-n``c-p选择输入命令历史记录里的前后字符串
  • c-y创建一个新文件及他的父目录
  • c-z用来标记多个文件，然后用c-o打开它们
  • ?or:help ctrlp-mappings可以调出帮助
  • ..可以到上一级的文件夹
  • 打开一个文件后:25可以跳到25行
  • 当打开多个文件时，在每个文件中:diffthis可以对比最多前四个文件的不同
