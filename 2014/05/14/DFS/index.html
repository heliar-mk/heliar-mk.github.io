DFS

By Kai Guan

5月 14 2014 更新日期:7月 13 2014

文章目录

DFS即Distributed File System（分布式文件系统）的缩写.

组成部分

  • DFS Namespace

让你可以把不同服务器(在不同的site内)的共享文件夹合在一起,构成一个或者多个逻辑结构的名称空间.用户看到的名称空间都是一个单独的共享文件夹,内部包含一系列子文件夹.真实的共享文件结构对用户是隐藏的,而且这种结构提供了容错功能同时具有自动链接用户本地共享文件夹的功能.(当本地具有所需的共享文件夹时,免去了通过WAN链接)

  • DFS Replication

通过LAN或者WAN在不同的服务器间同步文件夹.它采用Remote Differential Compression 协议,仅对那些从上次复制后发生变化的文件进行更新.
DFS Replication 可以和DFS Namespaces 一起使用也可以单独使用.
